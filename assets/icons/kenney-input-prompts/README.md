# Kenney Input Prompts

This directory contains a curated subset of the 128 px Xbox Series,
PlayStation Series, and Steam Deck glyphs from
[Kenney Input Prompts 1.5](https://kenney.nl/assets/input-prompts). They are
assembled into a runtime atlas by `packages/engine/ui_renderer.odin` and used by
the controller-mode Control Deck hint strip.

The subset covers the D-pad, shoulder buttons, south/east face buttons, and
light left-stick movement for each supported controller family.
