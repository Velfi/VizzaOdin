# Tabler Controller Icons

These SVGs are a small vendored subset of `@tabler/icons@3.44.0` from the
official Tabler Icons repository. The original outline SVGs are kept in
`outline/`. White-on-transparent PNGs are generated in:

- `png/48/`: 2x output from the 24px source.
- `png/96/`: 4x output for the large controller deck and 4K viewports.

## Controller mapping

- Look: `palette`
- Brush: `brush`
- Motion / Agents: `arrows-move`
- Awareness: `radar`
- Trails: `route`

- Birth: `sparkles`
- Capture: `video`
- Presets: `bookmarks`
- Pattern: `grid-pattern`
- Mask: `mask`

- Forces: `magnet`
- Physics: `atom`
- Population: `users-group`
- Advanced: `settings-cog`
- Field: `chart-grid-dots`
- Particles: `grain`

- Flow: `wind`
